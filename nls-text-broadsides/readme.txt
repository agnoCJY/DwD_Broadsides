Title: Broadsides printed in Scotland 1650-1910
Description: This dataset contains 1 plain text readme file, 1 inventory csv file and 1,727 plain text files.
Owner: National Library of Scotland
Creator: National Library of Scotland
Website: https://data.nls.uk/
DOI: https://doi.org/10.34812/a272-9961
Date created: 7/12/20
Rights: Items in this dataset up to 1854, and some later items, are free of known copyright and in the public domain. Items in this dataset from 1855 onwards are likely to be free of known copyright restrictions (No Known Copyright - https://rightsstatements.org/page/NKC/1.0/?language=en). In copyright items from this collection (also known as 'Word on the Street') have been removed to ensure this dataset can be made available openly.
Contact: digital.scholarship@nls.uk
